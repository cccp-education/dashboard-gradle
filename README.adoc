= dashboard-gradle 🟡

N3 Gradle plugin for visualizing workspace vision and progress.

== Structure

[source]
----
dashboard-gradle/
├── dashboard-plugin/        # Gradle plugin (independent build)
│   ├── AGENT.adoc           # Absolute rules
│   ├── .agents/             # Agent governance
│   ├── src/                 # Kotlin source code
│   └── build.gradle.kts     # java-gradle-plugin
├── build.gradle.kts         # Root consumer (dogfood)
├── settings.gradle.kts      # Root settings
└── gradle/                  # Wrapper + catalog
----

== Prerequisites

- JDK 24+
- Gradle 9.5+

== Development

[source,shell]
----
# Build the plugin
cd dashboard-plugin && ./gradlew publishToMavenLocal

# Exercise it from the root
cd .. && ./gradlew tasks
----

== Tasks

- `crawlDashboard` — crawls all `INDEX.adoc` and `SESSIONS_HISTORY.adoc` files and produces `build/dashboard/dashboard-data.json`
- `generateDashboard` — renders the static dashboard site (`index.html`, `styles.css`, DAG graph)
- `publishDashboard` — copies the generated site to `build/dashboard-publish`

== Testing

[source,shell]
----
./gradlew test          # unit + functional tests
./gradlew cucumberTest  # BDD scenarios
./gradlew check         # everything
----
