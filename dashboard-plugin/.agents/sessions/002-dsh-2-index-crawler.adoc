== Problèmes résolus

. `settings.gradle.kts` du subproject supprimé (ne doit pas exister — le root settings inclut le module)
. `YAMLMapper` → `ObjectMapper` (YAMLMapper écrit `---` dans un fichier `.json`)
. Trailing spaces après `|===` empêchant la regex de matcher les tables AsciiDoc
. `writeText()` sur Path remplacé par `Files.writeString()` (Kotlin extension non disponible)

== Décisions techniques

. Le modèle pivot est en JSON (pas YAML) car pivot d'échange standard
. Cucumber séparé des unit tests via `cucumberTest` task Gradle dédiée (excludeEngines)
. Utilisation de `cucumber-picocontainer` pour DI entre steps (pattern workspace)
. La tâche `crawlDashboard` écrit dans `build/dashboard/dashboard-data.json`
